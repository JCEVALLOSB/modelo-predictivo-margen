# Modelo predictivo de margen

## Descripción

Este repositorio contiene el código utilizado para desarrollar un modelo analítico orientado a estimar el margen operativo a partir de datos empresariales anonimizados.

El proyecto incluye preprocesamiento de datos, construcción de variables temporales, entrenamiento de modelos predictivos, evaluación de desempeño y comparación de alternativas metodológicas.

## Archivos principales

- Cevallos_Joan_Modelo_Predictivo_Margen.ipynb
- Cevallos_Joan_Modelo_Predictivo_Margen.py

## Herramientas utilizadas

- Python
- Google Colab
- pandas
- numpy
- scikit-learn
- XGBoost
- matplotlib

## Modelos evaluados

- Naïve temporal
- Media móvil de tres meses
- Regresión lineal
- Random Forest
- XGBoost

## Métricas de evaluación

- MAE
- RMSE
- MAPE
- R²

## Disponibilidad de datos

La base original no se publica porque contiene información empresarial confidencial. Por esta razón, el repositorio incluye únicamente el código fuente y la estructura metodológica necesaria para reproducir el flujo analítico con una base anonimizada equivalente.

## Instrucciones de reproducción

1. Abrir el notebook en Google Colab.
2. Cargar una base anonimizada con la estructura requerida.
3. Ejecutar las celdas en orden.
4. Revisar las etapas de preprocesamiento, construcción de variables, entrenamiento de modelos y comparación de métricas.
5. Comparar los modelos evaluados mediante MAE, RMSE, MAPE y R².
